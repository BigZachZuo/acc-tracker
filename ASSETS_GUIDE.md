# 图片素材命名指南

为了确保网站能正确显示图片，请严格按照以下文件名命名你的素材，并放入对应的文件夹中。

## 1. 文件夹结构

请确保你的项目根目录下有以下结构：

```
public/
└── assets/
    ├── ui/          (界面图标)
    ├── hero/        (首页轮播大图)
    └── tracks/      (赛道地图)
```

## 2. 界面素材 (`public/assets/ui/`)

| 文件名 | 说明 | 格式推荐 |
| :--- | :--- | :--- |
| **`logo.png`** | 网站左上角 Logo | 透明背景 PNG, 白色文字 |

## 3. 首页轮播图 (`public/assets/hero/`)

| 文件名 | 说明 | 格式推荐 |
| :--- | :--- | :--- |
| **`slide-1.jpg`** | 轮播图第1张 | JPG, 1920x1080 |
| **`slide-2.jpg`** | 轮播图第2张 | JPG, 1920x1080 |
| **`slide-3.jpg`** | 轮播图第3张 | JPG, 1920x1080 |
| **`slide-4.jpg`** | 轮播图第4张 | JPG, 1920x1080 |
